================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Python not found"
Fix: Restart your computer after installing Python, then try again

Problem: "pip install failed"
Fix: Check your internet connection, then run RUN_ME_FIRST.bat again

Problem: "No API key"
Fix: Edit .env file and add: OPENAI_API_KEY=sk-your-key

Problem: "Playwright browser not found"
Fix: Run this command manually:
     venv\Scripts\activate
     playwright install chromium
