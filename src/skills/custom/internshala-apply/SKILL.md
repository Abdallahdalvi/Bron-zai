# Internshala Fast Apply

## Description
A programmatic job applier for Internshala. It automatically navigates, checks your recent application history, filters new internships by profile, navigates to the first match, and starts the application. If there are custom assessment questions, it fills standard fields and pauses, letting the user fill the cover letter/custom fields manually.

## Trigger Phrases
- run internshala applier for [profile]
- apply to internshala jobs for [profile]
- automator apply internshala [profile]

## Required Tools
- navigate_page
- click
- fill
